# Competitive Landscape & Differentiators

| | GCash / Maya | Wise / Remitly | Coins.ph | **Sagip** |
|---|---|---|---|---|
| Instant P2P send | Yes | Cross-border | Yes | Yes (gift links) |
| Works without receiver installing an app | No | Partially | No | **Yes — link/QR only** |
| Conditional "if something happens to me" money | No | No | No | **Yes — core product** |
| Scheduled/occasion money (opens on a date) | No | No | No | Yes |
| Tuition released per term | No | No | No | Yes |
| Paluwagan with receipts | No | No | No | Yes (invite-only) |
| User-exportable keys (no lock-in) | No | No | Partial | Yes |
| Fees on rails | Interchange | FX spread | Spread | ~zero (Stellar) |

**Positioning:** not "another wallet." Sagip is the *continuity layer* for the family's
money — everything the big wallets treat as an edge case (death, silence, emergencies,
group trust, future dates) is our core. The practical takeaways we *did* adopt from
competitors: review-before-confirm screens (GCash), reference numbers on everything
(remittance apps), and plain-language receiver flows (Wise). What we deliberately did
not copy: cash-in/cash-out agent networks (partner instead), social feeds, crypto
trading, and loans.
